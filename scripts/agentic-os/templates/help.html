Support bundle

Builds a tarball under agentic-os-runtime/support-bundles/ containing a redacted copy of your config, doctor --sut --models --docker output, the tail of recent events, the last run's artifacts, and any bug notes on disk. Useful when filing an issue or asking for help.

Review the bundle before sending. Config secrets are redacted by a conservative leaf-key denylist (api_key, token, password, …), but other operator data (events, run artifacts, bug notes) is included verbatim. The denylist is conservative, not exhaustive.
